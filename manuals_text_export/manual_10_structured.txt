# TechniSat DIGIT XPS Operating Instructions

## Features

- With a PCMCIA interface suitable for Premiere for recording a CI module and integrated CONAX decryption system.
- For receiving free and encrypted (e.g. CONAX, VIACCESS, IRDETO, SECA etc.) satellite programs.
- With motor control and software update via satellite.

## Software Installation

1. After you have made the antenna settings, the digital receiver checks whether new software is available for the device via satellite. If new software is available, you can automatically carry out a software update.
2. Select the Update field with the left/right arrow keys and confirm with OK. (Image 5-12) The new software will now be downloaded for your device. This process may take a few minutes.
